# SwiftSFML

SFML bindings for Swift.

https://github.com/user-attachments/assets/49c78a27-45dd-4728-99e0-f27c850c7ef1

Has been tested with:

- [SFML v3.0.2](https://github.com/SFML/SFML/tree/3.0.2).
- [ImGui v1.92.4](https://github.com/ocornut/imgui/tree/v1.92.4).
- [ImGui-SFML v3.0](https://github.com/SFML/imgui-sfml/tree/v3.0).

Supported platforms:

- macOS 13 and higher.

## Prerequisites on macOS

- Install [Homebrew](https://brew.sh/).
- Install SFML using Homebrew: `brew install sfml@3.0.2`.
